# LSTM Partial Reset Research
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)  [![CI](https://github.com/afuckingco/lstm-partial-reset/actions/workflows/ci.yml/badge.svg)](https://github.com/afuckingco/lstm-partial-reset/actions)


Consolidated collection of periodic partial-reset LSTM research projects, each in its own subdirectory with independent history preserved via git subtree.

## Projects

- **energy-reset-lstm/** — Energy consumption forecasting with periodic partial reset LSTM
- **stock-reset-lstm/** — Stock price prediction with periodic partial reset LSTM
- **thesis-inertia-lstm-reset/** — Stress prediction with periodic partial reset LSTM (thesis project)

Each subdirectory retains its own README, LICENSE, and CI configuration.
