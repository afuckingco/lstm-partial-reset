# lstm-partial-reset

LSTM time-series forecasting with partial-reset dynamics. Experiment tracking, config-driven training, and evaluation.

## Stack

- Python 3.11+
- PyTorch / scikit-learn
- See `requirements.txt` or `pyproject.toml`

## Quick Start

```bash
git clone https://github.com/afuckingco/lstm-partial-reset.git
cd lstm-partial-reset
pip install -r requirements.txt
```

## CI

GitHub Actions — lint (ruff), build (py_compile). Every push triggers a run.
